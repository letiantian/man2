# man2

This tool is designed to get the usage of commands quickly. 

这个工具用于更快地查找命令使用方法。

## Install man2

```plain
$ npm install man2 -g
```

## Install documents

```plain
$ git clone https://github.com/letiantian/man2-docs.git ~/.man2-docs
```

If it is neccessary to update documents, run:
```plain
$ cd ~/.man2-docs
$ git pull origin master
```

## Usage
```plain
$ man2 ls
```
